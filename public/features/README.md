# public/features/

Hero images for magazine features go here. Reference them in MDX frontmatter:

```yaml
---
title: "Notes on Tokyo"
image: "/features/tokyo-hero.jpg"
---
```

## suggested specs

- ~1600×2000 (portrait) or 2000×1400 (landscape)
- JPEG, 80–85% quality
- One image per feature, the *best* image. Quantity dilutes; one strong photograph carries an entire article.

You can also reference images inline inside MDX bodies using standard markdown:

```mdx
![A picture of the Shimokitazawa thrift shop](/features/tokyo-thrift.jpg)
```
